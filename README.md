# Финальный проект 1 семестра

REST API сервис для загрузки и выгрузки данных о ценах.

## Требования к системе
Установленны Go и PostgreSQL

## Установка и запуск

Из корня репозитория запустить скрипт prepare.sh  
`./prepare.sh`

Затем запустить скрипт run.sh  
`./run.sh`

## Тестирование

Для тестов запустить из корня репозитория tests.sh  
`./tests.sh 1`
